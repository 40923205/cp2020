15 Reverse Word Order Solutions

Write a program (using functions!) that asks the user for a long string containing multiple words. Print back to the user the same string, except with the words in backwards order. For example, say I type the string:
編寫一個程序（使用函數！），要求用戶提供包含多個單詞的長字符串。向用戶打印相同的字符串，但單詞順序相反。例如，說我鍵入字符串：
My name is Michele
Then I would see the string:
我叫米歇爾 然後我會看到字符串：
  Michele is name My
shown back to me.
米歇爾是我的名字 顯示給我。
Discussion 討論區
Concepts for this week(本週的概念):
1. More string things 更弦的東西
Python has a lot of interesting things you can do with strings. I will show a few here, but you can see many more methods that may or may not be useful at the official Python documentation about the string format.
Remember that strings are lists.
Python有很多有趣的事情，您可以使用字符串來完成。我將在此處顯示一些方法，但是您可以在關於字符串格式的官方Python文檔中看到更多可能有用或可能沒有用的方法。 請記住，字符串是列表。
You can “split” or tear apart strings based on a given set of characters. For example:
您可以根據給定的字符集“拆分”或拆分字符串。例如：
  teststring = "this is a test"  result = teststring.split("t")
測試字符串 = "這是測試" 結果 = teststring.split("t")
And at the end, result will contain the list(最後，結果將包含列表):
  ['', 'his is a ', 'es', '']
Instead of "t", you can write any character you want. If you do not include any character, it means “split on all whitespace”:
您可以寫任何所需的字符，而不是“ t”。如果您不包含任何字符，則表示“在所有空格上分割”：
  teststring = "  this      has a lot    of   spaces and    tabs"  result = testring.split()
測試字符串 = “這有很多空格和製表符” 結果 = testring.split()
Then result contains (然後結果包含):
  ['this', 'has', 'a', 'lot', 'of', 'spaces', 'and', 'tabs']
2. Joinging Strings (連接字符串)
You can also relatively easily “join” or “smush” strings together:
您還可以相對容易地將字符串“連接”或“混入”：
  listofstrings = ['a', 'b', 'c']  result = "**".join(listofstrings)
字符串列表 = ['a', 'b', 'c'] 結果 = "**".join(listofstrings)
Then result will contain the string(然後結果將包含字符串):
  a**b**c
